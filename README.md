# Text Analyzer 📊

A simple web-based text analysis tool that provides detailed statistics about your text input.

## Features ✨

- 📝 Word count
- 🔤 Character count (with and without spaces)
- 📄 Sentence count
- 📑 Paragraph count

## Demo 🌐

Experience the tool live: [https://itsiamdev.github.io/Text-Analyzer/](https://itsiamdev.github.io/Text-Analyzer/)

## Usage 🚀

1. Paste or type your text into the input area.
2. Click the "Analyze" button.
3. Instantly see comprehensive text statistics displayed below.
